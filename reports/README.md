# Reports Directory

This directory contains generated analysis reports and their supporting data.

## Structure

### Markdown Reports (Root Level)
- `ANALYSIS_REPORT.md` - Comprehensive analysis across all categories
- `SIKA_ANALYSIS_REPORT.md` - PRODUCTOS SIKA detailed analysis (English)
- `REPORTE_SIKA_ESPANOL.md` - PRODUCTOS SIKA detailed analysis (Spanish)
- `DEPOSITO_TRUJILLO_INVESTIGATION.md` - Internal operations investigation
- `SIKA_PROVIDER_VERIFICATION.md` - Provider analysis report
- `TESTING_SUMMARY.md` - Testing results summary

### `/data/` - JSON Data Files
Raw data output from analysis scripts:
- `analysis_report.json` - General analysis raw data
- `sika_analysis_report.json` - SIKA analysis raw data

## Generating Reports

Reports are generated by scripts in `/scripts/reports/`:

```bash
# Generate English SIKA report
python scripts/reports/generate_sika_report.py

# Generate Spanish SIKA report
python scripts/reports/generate_sika_report_es.py

# Generate general analysis report
python scripts/reports/generate_report.py
```

## Report Updates

Reports are automatically generated when the corresponding analysis scripts are run. The JSON data files in `/data/` are used as input for report generation.
